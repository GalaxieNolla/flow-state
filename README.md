## Setup
Install fonts before running: open all files in `visuals/fonts/` and click Install Font

## Debugging
There are commented out lines with a # debug or # for debugging note. Uncomment out as needed to troubleshoot in the terminal.

## Distractions
In monitor.py there are list of distractions and exceptions. Update based on what you consider 
distractions or exceptions; e.g. if you use facebook or other sites as distractions, place them in the distraction list.
